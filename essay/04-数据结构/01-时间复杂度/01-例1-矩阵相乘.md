例1：两个矩阵相乘

    ``````javascript

        for(i=1; i<=n; ++i) {
            for(j= 1; j<=n; ++j) {
                c[i,j] = 0;
                for(k = 1; k<=n; ++k) {
                    c[i,j] += a[i,k] * b[k,j];
                }
            }
        }

    ``````

    .这个算法的控制结构是3层循环
    .这个算法的元操作有： 赋值、相加、相乘，其中乘法操作是算法的基本操作。

    结论：
        .乘法的运算次数，以作为算法的时间复杂度的衡量: n³
        时间复杂度： O(n³)
