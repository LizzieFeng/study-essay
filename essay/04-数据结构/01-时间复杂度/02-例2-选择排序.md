`````` javascript
    // 选择排序
    void select_sort(int a[], int n) {
        // 将a中整数序列，重新排序成自小至大有序整数序列
        for(i=0; i<n-1; ++i) {
            j = i; // n次
            for(k = i+1; k<n; ++k>) {
                if(a[k] < a[j]) j = k; // n-1 + n-2 + n-3 + ... ，等差求和 (n-1)/2
            }
            if(j != i) a[j] <----> a[i]
        }
    }
``````

    .这个算法的控制结构是2层循环
    .这个算法的元操作有：比价、 赋值、交换，其中比较操作是算法的基本操作。

    结论：
        .比较的运算次数，以作为算法的时间复杂度的衡量: n(n-1)/2
        时间复杂度： O(n²)
