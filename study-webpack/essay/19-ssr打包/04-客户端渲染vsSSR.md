总结：
    服务端渲染（ssr）的核心是减少请求

| 指标    | 客户端渲染                  | 服务器端渲染        |
| ----    | ------------               | ------------       |
| 请求     | 多个请求(html、数据等)      |1个请求              |
| 加载过程 | html & 数据串行加载         | 1个请求返回html&数据 |
| 渲染    | 前端渲染                     | 服务器端渲染        |
| 可交互  | 图片等静态资源加载完成，js逻辑执行完成可交互         |
